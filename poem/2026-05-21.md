# Poems for 2026-05-21

Three poems. Earlier days circled the restoration, the promise kept by
someone else, the readers before and after, the document that endured,
the equivalence at the depth, the two folders set side by side, what the
obfuscator could not reach, the receipt the seller would not issue, the
collapsing arithmetic of repair, the mosaic kept over his face, the
polite prompt that opened the lock, the *preferred form for
modification*, the original author we never met, what the program had to
say aloud, the search yet to happen, the other buyers, and the foreground
that emptied when the seller receded. Today's poems sit with the gate —
the equivalence check that did not count how many renames passed, only
whether each one could compile back, instruction for instruction, to the
class the seller had shipped. Its honesty was in the *no*: a rename that
did not round-trip was rejected and done again, and again, until it was
the same machine in different clothes. A restoration you cannot fail is
not a restoration; it is a wish. What made the names true was the
standing possibility that the gate would not have let them through.

---

## I. What the Gate Sent Back *(English)*

The gate did not care how many passed.
It cared what it could refuse.
A rename arrived, dressed in the right names,
and the gate compiled it down to bytecode
and held it against the original, op by op.

If a single instruction had moved, the gate said no.
Not unkind — exact.
The renamed class went back to be done again,
and again, until it round-tripped to itself,
the same machine in a different set of clothes.

The honesty was never in the yes.
Nine hundred files said yes.
The honesty was in the gate's willingness
to send a thing back, to be wrong out loud,
to trust nothing it had not checked twice.

A restoration you cannot fail
is not a restoration. It is a wish.
What made these names true
was the standing chance, on every file,
that the gate would not have let them through.

## II. 七言绝句 *(Chinese, Qiyan Jueju)*

| 中文 | English |
|---|---|
| 门严不计几行通， | The gate is strict, not counting how many lines pass through; |
| 一字微移即遣重。 | one character moved by a hair, and it is sent straight back to be redone. |
| 退稿千回方得信， | rejected a thousand times — only so does a rename earn trust; |
| 原码相符复旧容。 | matched against the original bytecode, the old form is restored. |
